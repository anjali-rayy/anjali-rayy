## 🚀 Featured Projects

### 🔍 [Image Quality Analyzer](https://github.com/anjali-rayy/image-analyzer) · [Live Demo](https://image-analyzer-nmri.onrender.com)
Full-stack image analysis tool built in **Go** — detects blur, exposure, noise, color profile & EXIF metadata in real time.
- 5 analyzers run **concurrently via goroutines** for fast processing
- Solved Android Chrome's EXIF stripping bug with client-side extraction
- Batch processing up to 20 images, CSV export, Dockerized & deployed
- **Stack:** Go · net/http · goexif · Docker · Vanilla JS

---

### ✍️ [Scribly](https://github.com/anjali-rayy/Scribly)
Blogging platform inspired by Medium with tag-based post recommendations.
- Firebase auth, Firestore real-time DB & React Context API for global state
- Tag-based engine that suggests posts based on reading history
- **Stack:** React · Firebase · Tailwind CSS · Vite

---

### 🏡 [CasaTrip](https://github.com/anjali-rayy/CasaTrip)
Travel stay listing app with interactive location maps.
- Mapbox integration for geo-location mapping of listings
- Full CRUD with MongoDB backend and EJS-rendered views
- **Stack:** Node.js · MongoDB · EJS · Mapbox
