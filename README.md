## Projects

**[Image Quality Analyzer](https://github.com/anjali-rayy/image-analyzer)** &nbsp;·&nbsp; [Live Demo](https://image-analyzer-nmri.onrender.com)

Real-time image inspection tool built in Go. Detects blur via Laplacian variance, noise, exposure, and extracts EXIF camera metadata. All 5 analyzers run concurrently using goroutines. Supports batch processing of 20 images, CSV export, and is Dockerized and deployed.

`Go` `net/http` `goroutines` `goexif` `Docker`
